### Solidity Course

1. Install and start [Ganache](https://www.trufflesuite.com/ganache) as a local Ethereum test node.
2. Install truffle
    ``` Shell
    npm install -g truffle
    ```
3. Install Nodejs dependencies
    ``` Shell
    npm install
    ```
4. Compile contracts
    ```
    truffle compile
    ```
5. Running migrations
    ```
    truffle migrate
    ```
6. Run a demo script
    ```
    truffle exec mytoken.js
    ```
